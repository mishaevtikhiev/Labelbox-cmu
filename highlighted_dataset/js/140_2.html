I think this is the best answer here so far. I'd just like to add that you can add parameters to `fix_spaces` using an anonymous function, like so: `fix_spaces(text, _) {
  return text.replace(/ /g, _);
}`. That way you can pass any number of parameters (to fix any number of spaces). Also, you can just use `text.replace(/ /g, "_")` by itself.
